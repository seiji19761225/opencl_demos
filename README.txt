This directory contains OpenCL demo programs.
The author confirmed these demo programs work appropriately on Linux with following OpenCL implementations:
  NVIDIA OpenCL
  Intel OpenCL

These demo programs use libpixmap for image I/O.
Before trying to compile these demo programs, you need to install libpixmap library.
You can download libpixmap from here:
  https://github.com/seiji19761225/libpixmap
